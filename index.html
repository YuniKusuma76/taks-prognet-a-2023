Kumpulan Tugas - Tugas Pemrograman Internet

01. Tugas JavaScript

Menampilkan form untuk menginputkan biodata dengan dilengkapi dengan penggunaan seluruh form element html yang ada. Tugas JavaScript ini dilengkapi dengan fitur validasi terhadap data yang diberikan oleh user.

02. Tugas PHP

Pada form yang sudah dibuat, user akan menginputkan data ke dalam form biodata tersebut. Kemudian, hasil inputan form biodata tersebut akan ditampilkan menggunakan PHP dan metode POST atau GET.

03. Tugas Database MYSQL

Membuat koneksi ke database Mysql didalam form biodata untuk digunakan mengirim dan menyimpan data inputan user ke dalam database. Kemudian, dapat melakukan CRUD, yaitu Insert, Select, Update, dan Delete data dalam database.
